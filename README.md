# Проект тестирования BooksCollector

## Описание проекта
Тестирование класса BooksCollector, который позволяет управлять коллекцией книг, устанавливать жанры и добавлять книги в избранное.

## Реализованные тесты

1. **test_add_new_book_add_two_books** - проверка добавления двух разных книг
2. **test_add_new_book_with_valid_name** - проверка добавления книги с валидным названием
3. **test_add_new_book_with_long_name** - проверка, что книга с названием >40 символов не добавляется
4. **test_set_book_genre** - проверка установки жанра книги
5. **test_get_books_with_specific_genre** - проверка получения списка книг по жанру
6. **test_get_books_for_children** - проверка, что книги с возрастным рейтингом не попадают в список для детей
7. **test_add_book_in_favorites** - проверка добавления книги в избранное
8. **test_delete_book_from_favorites** - проверка удаления книги из избранного
